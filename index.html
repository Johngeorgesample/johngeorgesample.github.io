Why stabby.js?

HTML is great for declaring static documents, but it falters when we try to use it to repeatedly and violently stab the user when they least expect it. stabbyJS is a fully extensible toolset that abstracts away most recognizable languages and markups and allows the user to manipulate the DOM through asynchronous multithreaded stabbings.

SUX

The surprise user xperience (SUX) philosophy details the importance of keeping users on their toes using unexpected results. Ranging from simple hover effects to radically altering the site, these results will keep the user guessing as to what may happen next.

Development

Because stabbyJS is an open source gift to the community, we want it to constantly evolve and grow. Developers are encouraged to create their own unique ways to stab users and branch the source. With more developers tying to stab users, together we can make the world a better place.
